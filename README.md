# Cassia Bake Studio

A modern website for Cassia Bake Studio built with Vite.

## Getting Started

1. Update Node.js to version 18 or higher.

2. Install dependencies: `npm install`

3. Run the development server: `npm run dev`

4. Open http://localhost:5173 in your browser.

## Sections

- Home

- About

- Menu

- Gallery

- Contact

## Build

`npm run build`

## Preview

`npm run preview`
